# Probe 1c: Symmetry Group of the Marked Cube

## 1. Full Ambient Group S₂ × S₃ × S₅ (order 1440)

Stabilizer order: 4

### All stabilizer elements:

  pol=id, lines=('H', 'P', 'Q'), elems=('Wood', 'Fire', 'Earth', 'Metal', 'Water')
  pol=id, lines=('H', 'Q', 'P'), elems=('Wood', 'Earth', 'Fire', 'Water', 'Metal')
  pol=swap, lines=('H', 'P', 'Q'), elems=('Wood', 'Water', 'Metal', 'Earth', 'Fire')
  pol=swap, lines=('H', 'Q', 'P'), elems=('Wood', 'Metal', 'Water', 'Fire', 'Earth')

### Structure analysis:

Elements with non-identity element permutation: 3
Elements with non-identity line permutation: 2
Elements with polarity swap: 2

### Detailed element descriptions:

  g0: pol=id    lines=id                    elems=id
  g1: pol=id    lines=(P Q)                 elems=(Fire Earth) (Metal Water)
  g2: pol=swap  lines=id                    elems=(Fire Water) (Earth Metal)
  g3: pol=swap  lines=(P Q)                 elems=(Fire Metal) (Earth Water)

### Is it cyclic/abelian?

Abelian: True
Element orders: [1, 2, 2, 2]
Not cyclic (no single generator)
Generated by g1 and g2

## 2. Algebraic Ambient Group Z₂ × S₃ × Aut(Z₅) (order 48)

Aut(Z₅) elements (as permutations of {0,1,2,3,4}):
  ×1: (0, 1, 2, 3, 4)
  ×2: (0, 2, 4, 1, 3)
  ×3: (0, 3, 1, 4, 2)
  ×4: (0, 4, 3, 2, 1)

Algebraic stabilizer order: 2

  pol=id    lines=id                    elems=×1 mod 5
  pol=swap  lines=id                    elems=×4 mod 5

Element orders: [1, 2]
Abelian: True

## 3. Interpretation

**The stabilizer is Z₂ × Z₂ (Klein four-group), order 4.**

Generated by two independent involutions:
- **g1: (P Q) swap + (Fire Earth)(Metal Water)** — swap lines P↔Q and simultaneously
  swap {Fire↔Earth, Metal↔Water}. No polarity change.
- **g2: polarity swap + (Fire Water)(Earth Metal)** — swap polarities and simultaneously
  swap {Fire↔Water, Earth↔Metal}. This is complement + Z₅ negation (×4 mod 5).

Their product g3 = g1·g2 combines both: swap polarity, swap P↔Q, apply (Fire Metal)(Earth Water).

All four elements are involutions (order 2), and the group is abelian — the Klein four-group V₄.

**Key observations:**
1. Wood is FIXED by every symmetry — it's the unique invariant element (Z₅ = 0, the kernel).
2. g2 (complement + negation) is the ONLY symmetry in the algebraic ambient group (Z₂ × S₃ × Aut(Z₅)).
   g1's element permutation (1 2)(3 4) is NOT in Aut(Z₅) — it's a non-multiplicative involution.
3. H (the 互 kernel line) is fixed by all symmetries — only P and Q can swap.
   This reflects H's structural distinction: the self-complementary pair {震,巽} both map to Wood,
   giving H a unique type-0 character that P and Q don't share.
