## 6. Key Findings

### Finding 1: Planet = f(palace, rank) — fully determined

The planet element is a **deterministic function** of (palace, rank). Given which palace a hexagram belongs to and its rank within that palace, the planet is completely determined.

### Finding 2: 生-cycle structure

Within each palace, the planet element advances through the **生 (generation) cycle** — one step per rank:

```
Wood → Fire → Earth → Metal → Water → Wood → Fire → Earth → ...
```

This is confirmed for 6 of 8 palaces (Qian, Kun, Zhen, Kan, Li, Gen) with **perfect 生 succession** across all 8 ranks.

### Finding 3: Anomalies

**Xun ☴ (KW#57 巽):** Text contains explicit lacuna marker `缺`. The 五星 passage is missing from the source. Based on the 生-cycle pattern (rank 1 = Metal), the missing planet should be **鎮星** (Saturn/Earth).

**Dui ☱ rank 2 (KW#45 萃):** Shows 熒惑 (Fire) where the 生-cycle predicts 嵗星 (Wood). Rank 3 (咸) also shows 熒惑 (Fire) — which IS correct for the cycle. This appears to be a **scribal error** (熒惑 written twice instead of 嵗星 then 熒惑). All other palaces follow perfect 生 succession.

### Finding 4: NOT redundant with palace element

Planet element matches palace element in only **14/63** cases (22.2%). Each palace cycles through **all five** elements across its 8 ranks — the planet is an independent cyclic coordinate, not a restatement of palace 五行.

### Finding 5: Low MI with any single feature

- H(planet_elem) = 2.3208 bits
- Best single predictor: **yang_count** (MI = 0.1744, only 7.5% of H)
- Because the planet cycles through all 5 elements within each palace, no single feature can predict it well. The information is distributed across the (palace, rank) pair.

## 7. Summary

### What 五星 encodes

The planet assignment in 京氏易傳 is a **生-cycle counter indexed by rank** within each palace:

1. Each palace starts its cycle at a specific element (offset varies by palace)
2. With each rank step (本宮→一世→二世→...→歸魂), the planet advances one step in the 生 cycle (Wood→Fire→Earth→Metal→Water→...)
3. Since 8 ranks cycle through 5 elements, the pattern repeats after 5 ranks (rank 5 = rank 0, rank 6 = rank 1, rank 7 = rank 2)

### Independence

- The planet is **not redundant** with palace element (22% match)
- It is **fully determined** by (palace, rank) — carries zero independent information
- It encodes **rank modulo 5** with a palace-specific phase offset
- Structurally: the planet is a **cyclic rank counter** dressed in 五行 language

### Information-theoretic status

The planet adds no new information to the system beyond what (palace, rank) already provides. It is a derived quantity — a deterministic function of the two primary coordinates of the 京房 八宮 system.
