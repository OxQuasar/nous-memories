========================================================================
  GL MAXIMIZATION THEOREM
========================================================================

  Theorem: For fixed N = q^m with q a prime power,
  |GL(m, F_q)| is maximized when q = 2.

  Proof: |GL(m, F_q)| = ∏_{i=0}^{m-1} (N − q^i).
  For fixed N, smaller q gives:
    1. More factors (m = log_q N is larger)
    2. Each factor N − q^i is larger (q^i smaller)
  Both effects increase the product. q = 2 minimizes q. ∎
