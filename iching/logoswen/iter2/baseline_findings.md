# Baseline & Comparison Findings (Threads B + F)

## Executive Summary

Three null models and one comparison ordering tested against the King Wen sequence. All results from actual computation (B1: 2000 samples, B2: 10,000 samples, F: deterministic).

**Core findings:**

1. **Eulerian property is a theorem** — 100% of orbit-paired sequences are Eulerian. Not an empirical finding; proved by degree-counting argument.
2. **S=2 absence is the most remarkable ordering property** — p = 0.012 under orbit-paired null. Equivalent to: no weight-5 bridges AND no weight-4 same-orbit bridges.
3. **Uniform matching per orbit is the most remarkable structural property** — KW uses the SAME generator mask for all 4 pairs within each orbit. Probability under random matching: (7/105)⁸ ≈ 4 × 10⁻¹⁰.
4. **Mask-orbit correspondence** — the mask for each orbit IS the orbit signature (sig ≠ 0) or OMI (sig = 0). This is the unique rule: "pair by swapping asymmetric line pairs."
5. **Fu Xi has zero orbit-consistent pairs** — its single-bit-flip structure is fundamentally incompatible with Z₂³ symmetry.
6. **Joint probability of all KW ordering properties: p < 10⁻⁴** — zero matches in 10,000 random orbit-paired sequences.

## Thread B1: Random Hamiltonian Paths on the 6-Cube

**Method:** 2000 random Hamiltonian paths on Q₆ via Warnsdorff heuristic (100% success rate, ~2200 paths/sec). Each path visits all 64 vertices exactly once, with consecutive vertices differing in exactly 1 bit.

### Why orbit-consistency is impossible on Q₆

**Parity argument:** All valid Z₂³ masks have EVEN Hamming weight:

| Mask | Weight |
|------|--------|
| id | 0 |
| O | 2 |
| M | 2 |
| I | 2 |
| OM | 4 |
| OI | 4 |
| MI | 4 |
| OMI | 6 |

A Hamiltonian path on Q₆ steps by single-bit flips (weight 1 = odd). Two consecutive hexagrams differ by exactly 1 bit. Therefore the XOR of any pair of adjacent hexagrams is a weight-1 mask, which is NEVER a valid Z₂³ mask.

**Structural implication:** The King Wen sequence is NOT a walk on the 6-cube's adjacency graph. Its transitions use the full weight spectrum {1,2,3,4,6} — these are steps on a different graph entirely (the Z₂³-coset graph).

### B1 Conclusion

B1 establishes that the KW structure lives in a completely different category from Gray code / Hamiltonian cube paths. The two are incompatible by parity. This rules out the null hypothesis "KW is a Hamiltonian path with incidental symmetry properties."

### The Eulerian Theorem (Proof)

**Claim:** Any orbit-paired sequence has an Eulerian bridge orbit walk.

**Proof:** Each orbit has exactly 8 hexagrams. Any valid Z₂³ mask partitions these into 4 disjoint pairs (each mask is a fixed-point-free involution). Therefore each orbit contributes exactly 4 pairs.

In any ordering of the 32 pairs:
- Each orbit appears as "pair orbit" exactly 4 times
- Bridge edges connect consecutive pair-orbits
- For any internal pair position, the orbit contributes 1 to out-degree and 1 to in-degree
- Only the first and last pairs create imbalance: +1 for the first orbit, -1 for the last

If first orbit = last orbit → Eulerian circuit (8.2% observed ≈ 1/8 expected)
If first orbit ≠ last orbit → Eulerian path (91.8% observed ≈ 7/8 expected)

Both satisfy Eulerian degree-balance. Connectivity guaranteed (31 edges on 8 nodes, average degree 7.75). **QED.**

### The S=2 Structural Decomposition

S=2 bridges come from exactly two sources (proven by exhaustive enumeration over all 4096 hexagram pairs):

| Bridge type | Weight | S value | Count (possible) |
|------------|--------|---------|-------------------|
| Weight-5, cross-orbit | 5 | S=2 always | 384 |
| Weight-4, same-orbit | 4 | S=2 always | 192 |
| Weight-4, cross-orbit | 4 | S=1 always | 768 |
| Weight-5, same-orbit | — | impossible | 0 |

Complete S-weight table:

| Weight | S=0 | S=1 | S=2 | S=3 |

**Therefore:** S=2 absence ⟺ (no weight-5 bridges) AND (no weight-4 same-orbit bridges).

KW achieves S=2 avoidance by:
1. Never placing hexagrams at Hamming distance 5 across consecutive pairs
2. Using self-loops (same-orbit bridges) only with weight 2 (B14: mask I) and weight 6 (B19: mask OMI), never weight 4

## Uniform Matching Discovery

**KW uses the SAME generator mask for all 4 pairs within each orbit:**

| Orbit signature | Orbit name | KW pair mask | 3-bit mask | Pattern |
|----------------|-----------|-------------|------------|---------|
| (0,0,0) | Qian | OMI | (1,1,1) | complement of sig |
| (0,0,1) | XChu | I | (0,0,1) | = sig |
| (0,1,0) | Shi | M | (0,1,0) | = sig |
| (0,1,1) | WWang | MI | (0,1,1) | = sig |
| (1,0,0) | Bo | O | (1,0,0) | = sig |
| (1,0,1) | Xu | OI | (1,0,1) | = sig |
| (1,1,0) | Zhun | OM | (1,1,0) | = sig |
| (1,1,1) | Tai | OMI | (1,1,1) | = sig |

**The Rule:** mask = orbit_signature for all orbits except Qian(000), where mask = OMI(111).

**Operational meaning:** "Pair each hexagram by swapping exactly the asymmetric line pairs." The orbit signature (o,m,i) tells which line pairs (L1-L6, L2-L5, L3-L4) differ. Using the signature as the pair mask means: correct the asymmetries. For Qian (all symmetric), use OMI = flip everything.

**Probability under random matching:**
- Each orbit has 7!! = 105 perfect matchings (complete graph K₈)
- 7 are uniform (all same mask, one per non-identity generator)
- P(uniform) = 7/105 = 1/15 per orbit
- P(all 8 uniform) = (1/15)⁸ ≈ 4 × 10⁻¹⁰
- P(specific KW mask assignment among uniform) = 720/7⁸ ≈ 1.25 × 10⁻⁴
- P(KW's exact pairing) = (1/105)⁸ ≈ 7 × 10⁻¹⁷

## Thread F: Fu Xi (Binary Ordering) Comparison

The Fu Xi ordering lists hexagrams 0–63 in binary value order.

### Pair Structure

All 32 Fu Xi pairs use mask `(1,0,0,0,0,0)` — flip bit 0 only. This is NOT a valid Z₂³ generator. Single-bit flips have odd Hamming weight (1); all Z₂³ generators have even weight. Fu Xi pairs cross orbits — they change the orbit's first component (o-bit).

### Bridge Structure

Fu Xi bridges follow the binary ruler sequence (carry pattern): XOR of (2k+1) and (2k+2). Weights cycle: 2,3,2,4,2,3,2,5,2,3,2,4,2,3,2,6,...

### Key Contrasts

1. **Orbit awareness:** KW pairs respect orbit structure (within-orbit). Fu Xi pairs violate it (cross-orbit by bit-0 flip).
2. **Directionality:** KW walks from Qian (all-symmetric, sig 000) to Tai (all-antisymmetric, sig 111). Fu Xi returns to origin (circuit).
3. **S=2 avoidance:** KW has zero S=2. Fu Xi has 2 (at the weight-5 carry positions B8, B24).
4. **Matching uniformity:** KW uses 7 distinct masks, one per orbit, determined by the orbit signature itself. Fu Xi uses 1 invalid mask for everything.
5. **Fu Xi Eulerian property:** also Eulerian (circuit type), but for a purely arithmetic reason — the binary carry pattern produces balanced orbit visits despite pairs crossing orbits.

### Fu Xi as a Structural Negative Control

Fu Xi is maximally regular and non-designed. It shares only the most basic structural features with KW (64 hexagrams, paired, uses all orbits equally). Every higher-level structural property fails:
- Zero orbit-consistent pairs
- S=2 present
- Weight-5 present
- No Qian→Tai directionality

This confirms that KW's structural properties are NOT artifacts of any general sequence property.

## Synthesis: Constraint Hierarchy

### Level 0: The space (given)
- 64 hexagrams ∈ {0,1}⁶
- 8 orbits of 8 hexagrams each (Z₂³ quotient structure)

### Level 1: Orbit-consistent pairing (the defining structural choice)
**Probability:** Not measurable under B1 — requires even-weight masks, which are impossible on Q₆ adjacency.
**Forces:** Eulerian bridge walk (theorem), all 8 orbits used, connected orbit graph.

### Level 2: Uniform matching per orbit (p ≈ 4 × 10⁻¹⁰)
KW uses the same mask for all 4 pairs in each orbit, with the mask determined by the orbit signature. This is not forced by Level 1 — random orbit-consistent sequences use mixed matchings.

### Level 3: Pair ordering constraints
Given the matching (Level 2), the ordering of pairs determines the bridge structure.

| Property | p-value (under Level 1) | Category |
|----------|------------------------|----------|
| Eulerian | 1.0 | **Forced** |
| Self-loops = 2 | 0.23 | Not significant |
| Prefix ≥ 6 | 0.13 | Marginal |
| Weight-5 absent | 0.046 | Significant |
| **S=2 absent** | **0.012** | **Significant** |
| **Qian→Tai** | **0.018** | **Significant** |
| **Joint (S=2 + Qian→Tai)** | **0.0002** | **Highly significant** |
| **All measured properties** | **< 10⁻⁴** | **Highly significant** |

### Level 4: Specific hexagram choices
Orientation of each pair (which hex first) and pair-slot assignments within orbits. Not measured in this analysis.

### The Genuinely Remarkable Properties

Ranked by statistical significance:

1. **Uniform matching rule (p ≈ 10⁻¹⁰):** The pairing itself — mask = orbit signature — is astronomically unlikely under random orbit-consistent pairing. This is the deepest structural choice.

2. **S=2 avoidance + Qian→Tai endpoints (p ≈ 0.0002):** Given orbit-paired sequences, the ordering that avoids S=2 while maintaining Qian-to-Tai directionality is rare. Only 2 of 10,000 random orbit-paired sequences achieved both.

3. **S balance (15/15 S=0/S=1):** Among the 1.22% of sequences achieving S=2 absence, the exact 15/15 balance is an additional constraint.

### What's NOT Remarkable

- **Eulerian property:** 100% under orbit-paired null. A theorem.
- **Self-loop count = 2:** 22.7% probability. Near the mode.
- **8 orbits used:** 100% always.
- **Orbit visit uniformity (4 per orbit):** Forced by orbit structure.
