## Part 2: Singleton-Forcing Window Verification

### Theorem

For complement-respecting surjections f: Z₂ⁿ → Z_p (p odd prime, p ≤ 2ⁿ−1):

> **Singletons are forced in every surjection ⟺ p > 2^(n−1)**

**Proof.** Let R = 2^(n−1) complement pairs. Each maps to one of S = 1 + (p−1)/2
slots: the zero-slot (both elements → 0) or a negation-pair slot {k, p−k}.
Surjectivity: m₀ ≥ 1 at slot 0, c_j ≥ 1 at each negation pair.
A singleton exists ⟺ some c_j = 1.
To avoid singletons: all c_j ≥ 2 → Σc_j ≥ 2·(p−1)/2 = p−1
→ m₀ = R − Σc_j ≤ R − (p−1) = 2^(n−1) − p + 1.
For m₀ ≥ 1: need 2^(n−1) − p + 1 ≥ 1, i.e., p ≤ 2^(n−1).
Contrapositive: p > 2^(n−1) ⟹ cannot avoid singletons. ∎

Conversely, if p ≤ 2^(n−1): set m₀ = 1, distribute R−1 pairs among
(p−1)/2 negation pairs with each c_j ≥ 2. Since R−1 ≥ 2·(p−1)/2 = p−1
when p ≤ 2^(n−1), this is achievable. So singleton-free surjections exist. ∎

### Key observation: partition correction at (3,5)

The prior analysis claimed the partition {2,2,2,1,1} is uniquely forced at (3,5).
This is **incorrect**. The correct enumeration shows two partition shapes:

- **[2, 2, 2, 1, 1]**: 192/240 surjections (80.0%)
- **[4, 1, 1, 1, 1]**: 48/240 surjections (20.0%)

The {4,1,1,1,1} partition arises when 2 complement pairs map to 0
(m₀ = 2), leaving the other 2 pairs to singly cover both negation pairs.

What IS correctly forced: **the presence of singletons**. Every surjection
at (3,5) has either 2 or 4 singletons. The minimum singleton count is 2.

### Structural pattern: shape count = cumulative partition numbers

**Theorem.** In the singleton-forcing regime (p > 2^(n−1)), the number of
distinct fiber partition shapes is exactly:

> **# shapes = Σ_{k=0}^{E} p(k)**

where E = R − S = 2^(n−1) − (p+1)/2 is the "excess" (complement pairs beyond
minimum needed) and p(k) is the number of integer partitions of k.

**Proof sketch.** Each fiber partition shape corresponds to a pair (e₀, λ)
where e₀ extras go to slot 0 (increasing m₀) and λ is a partition of E − e₀
distributing extras among the (p−1)/2 negation pair slots. In the forcing
regime, (p−1)/2 ≥ E, so the max-part constraint on λ is not binding,
giving p(E − e₀) choices for each e₀. Total: Σ_{e₀=0}^{E} p(E − e₀) = Σ_{k=0}^{E} p(k). ∎
