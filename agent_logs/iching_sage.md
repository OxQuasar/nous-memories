## Key Findings by Iteration (R167–R180)

### Iteration 17 — Q1 Phase 8: Opposition Direction Decomposition (R167–R170)

- **R167:** Trigram additive model R² ≈ 0.18, *below* null expectation of 0.226 (p ≥ 0.92 across all 3 models). Opposition direction does not decompose through trigram structure. This is the representation-theoretic test: the anti-symmetric subspace does NOT factor through trigram-pair irreducibles. The hexagram-level representation is irreducible.
- **R168:** Cross-model direction concordance ρ = 0.78–0.88 (Mantel on unit-vector cosine matrices). Procrustes R² = 0.85–0.95 at k=20. Opposition directions are text-intrinsic, not model artifacts.
- **R169:** Consensus participation ratio = 16.1 (vs 18.4 per-model). ~2 opposition dimensions are model-specific noise. The "true" dimensionality is ~16.
- **R170:** All algebraic groupings (Fano lines, 五行, basin) show null or anti-clustered direction alignment on residual centroids. Anti-clustering confirmed as residual-space artifact (disappears on raw centroids). No algebraic feature organizes opposition direction.
- **Q1 algebraic reduction path: CLOSED.** The ~16 dimensions are irreducibly non-algebraic.

### Iteration 18 — Q2 Proper: The Forcing Chain (R171–R175)

- **R171:** Minimum |S| satisfying Axioms 1-4 (polarity + dual evaluation + equivariance) is 6, not 8. A bare 6-element set with any FPF involution admits equivariant surjections to Z₅.
- **R172:** Group structure is NOT forced by Axioms 1-4. All FPF involutions on n elements are isomorphic as (S, σ) pairs. Group structure is additional data.
- **R173:** Adding the group axiom forces S = F₂ᵏ (Negation Uniqueness), then k ≥ 3 (size), then k = 3 (rigidity). F₂³ is the unique rigid group solution.
- **R174:** The forcing chain has exactly one gap: between "set with involution" and "group with translation." Composability is the one contingency.
- **R175:** The gap is irreducible. Four candidate fifth axioms (composability, line independence, self-referentiality, transition completeness) all restate group structure or rigidity. No axiom closes the gap without smuggling in the answer. Analysis of self-referentiality formalization (recursive evaluation) showed it reduces to restating the rigidity theorem as an axiom — circular.
- **Q2 Proper: CLOSED.** The structure is the minimum *rigid* implementation, and rigidity requires composability, which is not derivable from the concept of change.

### Iteration 19 — Resonance Test 3: Empirical Asymmetry (R176–R180)

## Methodological Notes

### Vulnerability (noted, not fixable)
All text-intrinsic findings depend on multilingual transformer embedding models trained on modern web corpora. Cross-model validation (3 models, 3 labs) mitigates model-specific artifacts but cannot address shared architectural assumptions. A radically different embedding approach (classical-Chinese-only, non-neural) could in principle yield different results. The tier system accounts for this: Tier 1b = "text-intrinsic as measured by current best methods."
